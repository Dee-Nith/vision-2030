# ENEXIO — Vision 2030

A scroll-driven web presentation for Enexio's Vision 2030 strategic meet, built around the metaphor of climbing Mount Everest via base camps.

## Live preview

Open `index.html` directly in Chrome, or run a local server:

```bash
python3 -m http.server 8765
```

Then visit `http://localhost:8765/`.

## Hosting on GitHub Pages

1. Push this repo to GitHub.
2. In your repo settings → Pages, select branch `main` and root folder `/`.
3. The site will be live at `https://<username>.github.io/<repo-name>/`.

## Stack

Pure HTML / CSS / vanilla JS — no build step, no dependencies. Single `index.html` file with inline styles and scripts.
